## Installation
In working directory ```cocktail-catalogue``` run in terminal:
```bash
docker compose build
docker compose run
```

- API is available at http://127.0.0.1:8000/api/
- Documentation at http://127.0.0.1:8000/api/docs/swagger/ for UI or http://127.0.0.1:8000/api/docs/ for YAML file
- Admin panel interface at http://127.0.0.1:8000/admin/ if you want to add database enteries
- Admin panel credentials: ```username: admin; password: admin```
